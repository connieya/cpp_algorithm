# n의 가장 오른쪽 0 켜기

## n |= n+1

n+1 은 n의 오른쪽 끝에서 연속된 1을 0으로 바꾸고, 처음 등장하는 0을 1로 바꿔서 만들 수 있다.

```c++
18 = 10010
19 = 10011

24 = 11000
25 = 11001
        
58 = 111010
59 = 111011

        
63 = 111111
64 = 1000000

71 = 1000111
72 = 1001000        

        
```

```c++
#include "bits/stdc++.h"

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int n = 0b00111011; // n = 0011 1011 (2)
    cout << n << '\n'; // 59
    n |= n+1; // n + 1 = 0011 1100 (2)
    cout << n << '\n'; // n = 0011 1111(2) , 63
}
```
