# n 의 가장 오른쪽  1 끄기

## n &= n-1

n-1 은 n 의 가장 LSB 비트와 그 오른쪽 비트를 뒤집는 것이다.

ex)
```c++
18 = 10010
17 = 10001
        
19 = 10011
18 = 10010
        
58 = 111010
57 = 111001        
```

그렇기 때문에 n 과 n-1 을 and 연산하면 
제일 마지막 1 (LSB) 가 0 이 되는 것이다.

```c++
#include "bits/stdc++.h"

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    // n의 가장 오른쪽 1 끄기
    int n = 0b00111010;
    cout << n << '\n';
    n &= n-1;
    cout << n << '\n';
}
```
