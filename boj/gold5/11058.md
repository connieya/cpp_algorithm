# BOJ 11058 크리보드


## solution 1

```c
#include "bits/stdc++.h"

using namespace std;
typedef long long ll;
ll dp[101] = {0,1,2,3,};

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int n;
    cin >> n;
    for (int i = 4; i <= 100; ++i) {
        for(int j=1; i -(j+2) >= 1; j++) {
            dp[i] = max({dp[i],dp[i-1]+1 , dp[i-(j+2)] *(j+1)});
        }
    }
    cout << dp[n];

}
```
