# BOJ 2436 공약수


## Solution 1

최대 공약수를 g , 최대 공배수를 l , 두 수를 A 와 B라고 하면

A = ga , B = gb , gcd(a,b) = 1 , ab = l / g 가 된다. 

곱이  l/g가 되면서 차는 최소인 두 수 a,b를 구하면 된다.

```c++
#include "bits/stdc++.h"

using namespace std;

int gcd(int a, int b){
    if (b==0) return a;
    return gcd(b,a%b);
}

int main() {
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int g,l ,a ,b , mx = INT_MAX;
    cin >> g >> l;
    int t = l/g;
    for (int i = 1; i*i <= t ; ++i) {
        if (t % i == 0 && gcd(i,t/i) == 1){
            a = i;
            b = t/i;
        }
    }
    cout << a*g << ' ' << b*g;
}
```
