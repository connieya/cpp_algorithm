# BOJ 10546 배부른 마라토너

## Solution 1

### unordered_set 사용

```c++
#include "bits/stdc++.h"

using namespace std;


int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int n;
    string name;
    cin >> n;
    unordered_set<string> s;
    for (int i = 1; i < 2*n; ++i) {
       cin >> name;
        if (s.count(name)) s.erase(name);
        else s.insert(name);
    }
    cout << *s.begin();
}

```
