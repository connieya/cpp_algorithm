# BOJ 11720 숫자의 합

## solution 1

```c
#include <cstdio>
int main()
{
    int n,x;
    int rst =0;
    scanf("%d",&n);
    while (n--){
        scanf("%1d",&x);
        rst+=x;
    }
    printf("%d\n",rst);
}
```

## solution 2

```c
#include "bits/stdc++.h"

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int n;
    string str;
    cin >> n >> str;
    int ans  = 0;
    for(char ch : str) ans += ch-'0';
    cout << ans;
}
```
