# BOJ 25304 영수증 

## solution 1

```c
#include "bits/stdc++.h"

using namespace std;

int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int res ,n;
    cin >> res >> n;
    for (int i = 0; i < n; ++i) {
        int a,b;
        cin >> a >> b;
        res -= a*b;
    }
    cout << (res ? "No" : "Yes");
}
```
