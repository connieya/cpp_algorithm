# BOJ 17427 약수의 합 2

## solution 1

```c++
#include "bits/stdc++.h"

using namespace std;
int main()
{
    ios_base::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    int n;
    long long answer = 0;
    cin >> n;
    for (int i = 1; i <=n; ++i) {
        answer += (n/i)*i;
    }
    cout << answer;
}
```
