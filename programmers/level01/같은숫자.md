# 프로그래머스 Lv.1 같은 숫자는 싫어

## solution 1

```c++
#include <bits/stdc++.h>

using namespace std;

vector<int> solution(vector<int> arr) 
{
    int prev = -1;
    vector<int> answer;
    for(auto a : arr){
        if(prev != a) answer.push_back(a);
        prev = a;
    }
    return answer;
}
```

## solution 2

### unique 키워드

```c++
#include <bits/stdc++.h>

using namespace std;

vector<int> solution(vector<int> arr) 
{
    arr.erase(unique(arr.begin(),arr.end()),arr.end());
    return arr;
}
```
