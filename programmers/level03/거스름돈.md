#  프로그래머스 Lv.3 거스름돈

## solution 1


```c
#include <bits/stdc++.h>

using namespace std;

int solution(int n, vector<int> money) {
    int answer = 0;
    vector<int> dp(n+1);
    dp[0] = 1;
    for(int m : money){
        for(int i=m; i<= n; i++){
            dp[i] += dp[i-m];
        }
    }
    return dp[n];
}
```
